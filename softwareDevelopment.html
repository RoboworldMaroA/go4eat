I built an app that makes a packing easy through an easy-to-use smart item allocation system. It allocates items to the appropriate category based on the user's activity and gender. It calculates the weight and number of items in the packed luggage. It allows you to add your own items and delete unnecessary ones. The colors in the app are well thought out and consistent. Google Map API is used to search destination, it allows to find practically every address in the world that is in Google database. Weather API is used to displays the temperature and an icon with the current weather status. The vision system allows the user to add elements using a camera. Application uses a cocoSSD model and Tensorflow.js. User can switch between cameras available on the device. The application is intuitive and uses the latest technology. In addition registration, login and administration panel is implemented.

Vision System


According to the project requirements as an optional requirement is ability to add user items by using the camera. Unfortunately, it was not possible to integrate the vision system into the application. The prototype application was built on an Ubuntu system and written in Python using Open CV libraries and SIFT functions. At the moment it is a separate application. It can be integrated into the main system in the next stage of the project. Application contains in the memory objects which the system recognizes and displays on the screen information what was found. I build training model that allows add an object to the system. System asking the user to take a photo, then the photo is stored in memory, and in the next phase the application is opened and can recognize whether the object is in the database or not.

Setup Environment

I used the instructions posted on pyimagesearch.com to install all dependencies.
Main dependencies installed on Ubuntu 20.04

To run the application, I created a bash script as shown in the photo below.

The largest window is the main window. There is a smaller blue square in this window. This is the place where we need to place the object we want to recognize.The smaller window on the left shows only what we see in this blue zone.

Press the space bar to take the photo. New window will appear with the photo that just have been taken. The system will ask you if you are happy with the result. If so, confirm by pressing esc or if not press space again and choose another frame.

The image will then be saved to your hard drive and object recognition will begin.

The other two objects stored in the database are a G-Shock watch and a pack of Oral-B toothpaste. They are displayed on the screen on the top left corner If you want to test the app, you can open the photo stored in the app and take a picture with your phone, then show the photo instead of the object - in this case the app works much more accurately.
