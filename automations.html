Industrial Robots and Automations

I had opportunity to use my coding skills while working as an automation engineer in the R&D department at Prodieco and at the Burnside Autocyl as a robots programmer.

Below are some of the projects I have completed

Machine for sorting tablets according to their thickness.

My task was to build a machine to sorting tablets. This machine was fitted just behind the exit from the eliminator broken tablets.

The tablet sorting device consists of eight rollers mounted on a guide.

Two DC stepper motors allow the rollers to move along the guide and thus create a gap through which tablets will pass to the other side if their size is within a set tolerance. An AC motor was used to rotate the rollers.

My responsibilities:

Assembly machine

Design the electrical wiring diagram

Build the electrical box

Write PLC program

Design and Program HMI touch panel

Test

Modifications

The device was controlled by a Mitsubishi FX5U. To create program for PLC and HMI used MELSOFT GX Works 3 and GT-DESIGNER 3.

The algorithm for determining the drop zone was created in Excel and converted to PLC.
